# DashGit

**WIP**

Dashboard to get a quick look over GitLab pending mr pipelines and jobs using gitlab api

## Setup

Create `.env.local` with:
```
VITE_GITLAB_BASE_URL
VITE_GITLAB_TOKEN
```

## Run

```
npm run dev
```
